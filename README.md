🧾 Project Name: BD Food Storage
Live Site: bd-food-storage.web.app

📌 Project Overview
BD Food Storage is a modern, responsive web application designed to help users efficiently track, manage, and monitor food items in their storage. The app focuses on organizing perishable items with expiration tracking, visual stats, alerts for expiring goods, and an intuitive user interface that supports both desktop and mobile devices.

It is a full-featured client-side application built with React 19 and optimized for performance and developer experience using Vite. Firebase is used for authentication, database storage, and deployment, making it a scalable and secure platform.

🚀 Key Features
✅ User Authentication via Firebase

📦 Food Item Listing with expiration date and details

⏰ Upcoming Expiry Alerts with countdowns

📊 Visual Stats with animated counters (react-countup)

📸 Carousel view for featured or recently added foods (react-flicking)

💡 Tooltips, toasts, and modals for better UX

🎉 Animated UI using Lottie, Awesome Reveal, and Typewriter effects

📱 Fully responsive and mobile-friendly
